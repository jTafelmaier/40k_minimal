Core rules

To start a match, players agree on:
  • the board space (including special zones like cover)
  • a starting player
  • a total army cost limit (example: 1000 points)
  • a game duration limit (number of turns).
Then, every player chooses unit instances of one faction, with a total points cost not exceeding the agreed limit. Only one unit per unit type can include less models than its model count limit.

Every unit starts the game in reserves (separate area, not on the board space). Every turn, each unit can once sequentially spend 2 AT to perform different actions, alternating between sides. All units including a model of type "character" may perform their actions before any other units.

At the end of all turns, a winning player is determined by the total points cost of units on the board space.

Action [2AT]
Deploy
This action can only be performed when this unit is in reserves.
This unit may be placed anywhere in the owner's deployment zone.
Action [2AT]
Setup deepstrike
Set up a deepstrike token on the board space.
When this unit performs the "Deploy" action: May place the unit as close as possible to the placed deepstrike token, and then remove that token.
Action [1AT]
Move
Every model in the unit may move up to its movement characteristic in centimeters, without moving over enemy models.
Action [1AT]
Attack
If possible, select one attack and one visible elegible enemy unit as "target unit", otherwise end the action.

Target unit: health points total are reduced by:
max(0, (#models * (damage)) - damage reduction).

Models in the target unit have to be destroyed until their count is equal to ceil(health points total / health points per model).
Unit
A collection of one or more identical models that act as one entity.
When a unit contains more than 1 model, every model not within 5 cm of any other model is immediately destroyed.
In cover
A unit is in cover if it is entirely within a cover object, and the attacking unit is not within 50 cm.
Destroy model
The controlling player must remove the model from the board space.

stealth
This unit is considered in cover while in a cover object and the attacking unit is not within 20 cm.
shielded
Enemy units that are either the closest to the target unit or within 20cm of the closest enemy unit, deal 0 damage if they would have dealt less than 1 damage.

Range: ⊹
Cannot attack units in cover.
Range: ⛶
Can only be used against targets within 20 cm.
Range: melee
Can only be used against targets within 5 cm.
volume
While targeting a unit including only one model, weapon damage is halved.
single

heavy
